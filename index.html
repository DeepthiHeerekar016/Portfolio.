My Project

Line Following Robot

The Line Following Robot is an Arduino-based project that autonomously follows a predefined path using infrared sensors. It detects and tracks lines on the ground by processing sensor data to control motor movements.

Doctor's Handwritten Recognition

The Doctor’s Handwritten Recognition project uses Python and machine learning to convert handwritten prescriptions into digital text. It applies image processing techniques to accurately interpret and extract information from handwritten notes.

A machine learning-based application that recommends suitable yoga poses based on user inputs such as body type, flexibility, and fitness goals. It helps users choose the right poses to improve health, balance, and posture effectively.
