My Project

Line Following Robot

I have developed a project called a Line Following Robot using Arduino. This project involves designing a robot that can autonomously follow a predefined path. It uses sensors to detect and follow lines on the ground, making it suitable for applications like warehouse automation or educational robotics. The robot's behavior is controlled by an Arduino microcontroller, which processes sensor inputs and adjusts the motors to stay on course. This project showcases my skills in electronics, programming, and robotics.

Doctor's Handwritten Recognition

I have developed a project called Doctor's Handwritten Recognition using Python, which focuses on converting handwritten prescriptions into digital text. This project utilizes various Python libraries and tools, including machine learning algorithms and image processing techniques, to accurately interpret and transcribe handwritten notes. By analyzing images of handwritten text, the system can extract relevant information and present it in a legible format, aiding in the digitization and organization of medical records. This project demonstrates my proficiency in Python programming, image processing.
